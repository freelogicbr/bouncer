## Reporting a Vulnerability

To report a security vulnerability, use [GitHub's private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability) feature.

## Security Notes

- Never commit `.env` files or credentials.
- Keep sensitive configuration local only.
- Use environment variables for secrets.
