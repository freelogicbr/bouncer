# CONTRIBUTING.md - Guia de Contribuição / Contribution Guide

## PT-BR / Português

Agradecemos o interesse em contribuir! Este projeto segue um modelo de coordenação via LLM.

### Regras de Contribuição

1. **Sugestões via Issue**: Proponha melhorias ou reporte bugs através de Issues.
2. **Sem Edição Direta**: A estrutura do projeto é gerida por modelos de IA coordenados pelo usuário.
3. **Padrão Bilíngue**: Todas as contribuições de documentação devem ser enviadas em PT-BR e EN.
4. **Segurança**: Garanta que segredos e dados sensíveis não sejam incluídos.

---

## EN / English

Thank you for your interest in contributing! This project follows an LLM-coordinated model.

### Contribution Rules

1. **Suggestions via Issues**: Propose improvements or report bugs through Issues.
2. **No Direct Editing**: The project structure is managed by AI models coordinated by the user.
3. **Bilingual Standard**: All documentation contributions must be submitted in both PT-BR and EN.
4. **Security**: Ensure that secrets and sensitive data are not included.
